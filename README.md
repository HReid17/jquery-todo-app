# jQuery To-Do App

A simple interactive to-do list built with jQuery to demonstrate core concepts such as DOM manipulation, event handling, and dynamic UI updates.

---

## Features

* Add new tasks
* Mark tasks as complete
* Delete tasks
* Prevent empty task submissions
* Add tasks using the Enter key

---

## jQuery Concepts Demonstrated

* **Selectors** – targeting elements in the DOM
* **Event Handling** – handling clicks and key presses
* **DOM Manipulation** – adding and removing elements
* **DOM Traversing** – using `.parent()` and `.siblings()`
* **Chaining** – performing multiple actions on elements
* **Event Delegation** – handling dynamically added elements

---

## How to Run

1. Clone the repository:

```
git clone https://github.com/HReid17/jquery-todo-app.git
```

2. Open the project folder

3. Open `index.html` in your browser
   *(or use Live Server in VS Code)*
